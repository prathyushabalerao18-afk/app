# Personal Expense Tracker

A lightweight single-page expense tracker for recording purchases, reviewing category totals, and monitoring recent spending.

## Features

- Add expenses with amount, date, category, payment method, and notes.
- Automatic dashboard totals for overall spending, leading category, and total entries.
- Category breakdown cards with proportional progress bars.
- Recent-expenses ledger with quick deletion.
- Responsive layout for desktop and mobile screens.

## Run locally

Because the project is a static site, you can open `index.html` directly in a browser or serve it with a small local web server:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.
